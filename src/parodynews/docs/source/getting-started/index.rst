===============
Getting Started
===============

Welcome to the Parodynews getting started guide! This section will help you install,
configure, and start using Parodynews for AI-powered parody news generation.

Prerequisites
=============

Before you begin, ensure you have:

* **Docker & Docker Compose**: For containerized development
* **Python 3.8+**: If running outside containers
* **OpenAI API Key**: For content generation
* **Git**: For version control
* **PostgreSQL**: For production database (or use SQLite for development)

What's Covered
==============

This guide covers everything you need to get started:

1. **Installation**: Set up the development environment
2. **Configuration**: Configure environment variables and settings
3. **Quickstart**: Run your first content generation
4. **First Content**: Create your first AI assistant and generate content
