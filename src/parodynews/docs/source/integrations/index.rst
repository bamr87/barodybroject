Overview
========

OpenAI Integration
------------------

Parodynews uses OpenAI's API for AI-powered content generation. See :doc:`openai`
for configuration details.

GitHub Integration
------------------

Publish generated content directly to GitHub Pages. See :doc:`github` for setup.

Docker Support
--------------

Run Parodynews in Docker containers for consistent development and deployment.
See :doc:`docker` for details.

Azure Deployment
----------------

Deploy to Azure Container Apps with Bicep infrastructure templates. See :doc:`azure`
for deployment guides.
