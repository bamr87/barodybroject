Overview
========

Parodynews provides a comprehensive workflow for AI-powered content creation:

1. **Configure Assistants**: Set up AI assistants with custom instructions
2. **Generate Content**: Use assistants to create parody news articles
3. **Review and Edit**: Review generated content and make adjustments
4. **Publish**: Publish content to your GitHub Pages site

Features
========

* Multiple AI assistant configurations
* Thread-based conversation management
* Template-based content generation
* JSON schema validation
* GitHub Pages integration
* Django admin interface
