========
Glossary
========

Terminology and definitions.

.. glossary::

   Assistant
      An AI configuration with custom instructions for content generation.
   
   Thread
      A conversation thread with an OpenAI assistant.
   
   Message
      A single message within a thread (user or assistant).
   
   Run
      An execution of an assistant on a thread.
   
   Content Item
      A piece of generated content (article, post, etc.).
   
   JSON Schema
      A schema definition for structured data validation.

(More terms to be added)
